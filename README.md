# LCS Grade Calculator

A web-based grade calculator for the Grade 11 Life and Career Skills subject, aligned with the Philippine Strengthened Senior High School Curriculum.

This tool is designed to help teachers and learners compute grades accurately based on subject requirements and weighting.

---

## Usage
Open the calculator in any modern browser. No installation required.
